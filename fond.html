Mettre l'image

On utilise "background-image: url("");".
Voyer par vous meme le rendu d'un fond fixe non répété

Fixer

On peut fixer le fond avec "background-attachment:" suivi de "fixed" ou de "scroll" (defile avec le fond, par defaut).

Repetition

Pour gerer la repetition du fond, on utilise "background-repeat:" avec :

  • "no-repeat" : le fond ne sera pas repete.
  • "repeat-x" : repete seulement sur l'horizontal.
  • "repeat-y" : repete seulement sur la vertical.
  • "repeat" : repete en mosaique (par defaut).

Position

On utilise "background-position:". Il faut qu'elle soit couplee avec "background-repeat: no-repeat" pour qu'elle soit interessante.

  • On donne deux valeurs (decalage a gauche et en bas) en pixels (px) par rapport a l'angle superieur gauche de la page

  • "top" pour en haut
  • "bottom" pour en bas
  • "left" pour a gauche
  • "center" pour centre
  • "right" pour a droite
  • Il est aussi possible de combiner ces mots

Combiner les proprietes

Si on utilise plusieurs proprietes on les rassemblent sous "background:"; et aussi plusieurs images de fond dans l'ordre d'affichage.
Exemple : "background: url("fond1.png") fixed no-repeat top right, url("fond2.jpg") fixed;"
Dans n'importe quelle ordre.

Transparence

On utilise "opacity:" avec une valeur de 0 (totalement trasparent) a 1 (totalement opaque, par defaut).
Mais cela sera applique a tout le contenu de cet element.
Si c'est pour juste jouer sur la transparence du fond, il faut plutot utiliser la notation RGBa . Exemple pour une transparence de 0.5: "background-color : rgba(255, 0, 0 ,0.5)"
