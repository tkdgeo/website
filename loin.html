Du site a l'application web

On utilise JavaScript pour charger de nouveaux elements sur la page sans avoir besoin de la recharger. C'est un langage de programmations qui ressemble au C/C++, Python, Ruby... contrairement au HTML et au CSS.

Avec HTML5

Voici une liste de technologies introduites en parallele de HTML5 :

  • Canvas : permet de dessiner au sein de la page web, al'interieur de la balise HTML "canvas". On peut ajouter des formes (carre, triangle...) mais aussi des images pour ensuite les manipuler. Cela permet ainsi de realiser de veritable jeu.
  • SVG : permet de creer des dessins vectoriels au sein de la page web. A la difference de Canvas, ils peuvent etre agrandis a l'infini. Un exemple de logiciel : Inkscape.
  • Drag & Drop : permet de faire "glisser-deposer".
  • File API : permet d'acceder au fichiers stockes sur la machine du visiteur. A utiliser, par exemple, en combinaison avec Drag & Drop.
  • Geolocalisation : pour geolocaliser le visiteur et lui proposer des services.
  • Web Storage : permet de stocker un grand nombre d'informations sur la machine du visiteur. Les informations sont hierarchisees comme dans une base de donnees.
  • Appcahe : permet de demander au navigateur de mettre en cache certains fichiers. Utile pour les applications web qui peuvent fonctionner meme en mode "hors ligne".
  • Web Sockets : permet des echanges plus rapides, en temps reel, entre le navigateur du visiteur et le serveur qui gere le site web (sorte d'AJAX ameliore).
  • WebGL : permet d'introduire de la 3D dans les pages web.
La plupart de ces technologies s'utilisent avec JavaScript.

Les sites web dynamiques

Ces langages s'execute sur le serveur alors que JavaScript s'execute sur la machine de l'utilisateur. Par exemple :

  • PHP
  • JEE (Java)
  • ASP .NET
  • Django (Python)
  • Ruby on Rails (Ruby)
Connaitre l'un de ces langages est indispensable pour traiter des formulaires "form" en HTML.
