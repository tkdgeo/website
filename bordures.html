Les bordures

Les bordures standard

La super-propriete "border" fonctionne comme "background", elle regroupe "border-bottom-width", "border-color", border-style"...
On peut utiliser jusqu'a 3 valeurs pour modifier l'apparence de la bordure :

  • la largeur : mettre la valeur en pixels (px)
  • la couleur : soit en nom, en hexadecimale ou en RGB
  • le type :
    • "none" : pas de bordure (par defaut)
    • "solid" : trait simple
    • "dotted" : pointilles
    • "dashed" : tirets
    • "double" : bordure double
    • "groove" : relief
    • "ridge" : autre effet relief
    • "inset" : effet 3D global enfonce
    • "outset" : effet 3D global sureleve

On peut choisir une bordure differente pour les bords :
  • "border-top" : bordure du haut
  • "border-bottom" : bordure du bas
  • "border-left" : bordure de gauche
  • "border-right" : bordure de droite
Il y a aussi des equivalents pour chaque detail :
  • "border-top-width" : epaisseur bordure du haut
  • "border-top-color" : couleur du haut
  • etc.
Ce sont aussi des super-proprietes.
Il est possible de modifier les bordures des images, des textes comme strong...

Les bordures arrondies

On utilise la propriete "border-radius:" suivi de la taille de l'arrondi en pixels (px).
Les arrondies se voient avec une bordure ou une couleur de fond.
On peut indiquer l'arrondi des 4 coins : "border-radius: 10px 5px 10px 5px;"
On commence en haut a gauche puis sens horaire.
Ou encore des courbes elliptiques : "border-radius: 20px / 10px;"

Les ombres

Les ombres des boites

La propriete est "box-shadow:", elle s'applique a tout le bloc et prend les 4 valeurs dans l'ordre suivant :

  1. le decalage horizontal de l'ombre en px
  2. le decalage vertical de l'ombre en px
  3. l'adoucissement du degrade en px
  4. la couleur de l'ombre
  5. facultative : "inset". Dans ce cas l'ombre est placee a l'interieur du bloc.
Exemple : box-shadow: 6px 6px 0px black;
Pour les navigateurs mobiles il faudra utiliser "-webkit-box-shadow"

L'ombre du texte

Avec "text-shadow:" avec les 2 décalages, adoucissement et couleur.
