Les balises de type block et inline

  • block : cree automatiquement un retour a la ligne avant et apres (div).
  • inline : est obligatoirement a l'interieur d'un block (span).
    • Quelques exemples

Les dimensions du type block

On dispose en CSS de "width" pour determiner la largeur (en px ou %) et de "height" pour la hauteur (en px ou %).
On peut aussi poser des dimensions min et max pour un bloc avec :

  • "min-width" : largeur min
  • "min-height" : hauteur min
  • "max-width" : largeur max
  • "max-height" : hauteur max
On peut ainsi combiner les deux.
Ca valeur par defaut est "auto". La page prend alors toute la place disponible.

Les marges

Il y a les marges interieurs et exterieurs :

On peut les modifier avec "padding" pour l'interieur et "margin" pour l'exterieur, les deux en pixels en general.
Remarque : les balises de type inline ont aussi des marges.
Elles s'applique au 4 cotes. Comme pour border, on a top, bottom, left, right pour les modifier une par une.
Exemple : "margin-top" et "padding-bottom". Il y a d'autres façons de spécifier les marges avec les propriétés margin et padding. Par exemple : margin: 2px 0 3px 1px; signifie « 2 px de marge en haut, 0 px à droite (le px est facultatif dans ce cas), 3 px en bas, 1 px à gauche ». Autre notation raccourcie : margin: 2px 1px; signifie « 2 px de marge en haut et en bas, 1 px de marge à gauche et à droite ».

Pour centrer les blocs il donner un largeur avec "width" puis indiquer des marges exterieures automatique avec "margin: auto;"

Quand ca depasse

Quand le bloc qu'on a definit est trop petit pour le texte on utilise "overflow". Cette balise peut prendre les valeurs suivantes :

  • "visible" : le texte sort sort du cadre mais reste visible (par defaut)
  • "hidden" : le texte exterieur n'est pas visible
  • "scroll" : barre de defilement sur le cote qui permet de voir le texte entier
  • "auto" : le nvigateur met des barres de defilement si c'est necessaire
De plus, la fonction "word-wrap: break-word;" permet de couper un mot lorsqu'il est trop long pour entre dans le bloc (comme un lien par exemple).
